# svagent-ui

svagent-ui is a component registry for building AI agent interfaces. It provides voice, chat, transcription, and visualization components for Svelte 5.

## Features

- **Provider-agnostic** — Wire up your own backend (ElevenLabs, OpenAI, Deepgram, custom, etc.)
- **Svelte 5** — Built with runes, snippets, and modern Svelte patterns
- **Tailwind CSS 4** — Styled with tailwind-variants and tailwind-merge
- **Accessible** — Built on bits-ui primitives
